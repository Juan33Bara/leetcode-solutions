# Approach
- Use the method **toLowerCase()** 

# Complexity
- Time complexity: O(n). The method must scan every character in the string to check and convert it.

- Space complexity: O(n). In java, string are immutable, so the method creates a new string.

# Code
```java []
class Solution {
    public String toLowerCase(String s) {
        return s.toLowerCase();
    }
}
```
